돈포겟(Don't Forget) 개인정보처리방침

최종 업데이트: 2026년 7월 16일

돈포겟(Don't Forget, 이하 "당사 앱")은 사용자의 개인정보를 소중하게 다루며, 관련 법령에 따라 다음과 같이 개인정보를 처리합니다. 당사 앱은 할 일을 배경화면·위젯용 이미지로 만들어 주는 메모 앱으로, 별도의 회원가입·로그인 없이 사용됩니다.

1. 당사 앱이 직접 수집·저장하는 정보

당사 앱은 사용자의 개인정보를 자체 서버로 수집하거나 전송하지 않습니다. 당사 앱은 자체 서버를 운영하지 않으며, 회원가입·로그인·클라우드 동기화 기능을 제공하지 않습니다. 사용자의 이름, 이메일, 전화번호, 계정 정보를 수집하지 않습니다.

사용자가 앱 내에서 입력한 할 일 목록, 메모, 테마·글꼴 등 표시 설정은 사용자 기기 내부의 로컬 저장소에만 저장됩니다.

2. 이미지 저장 기능

당사 앱은 사용자가 작성한 할 일 목록을 이미지로 만들어 기기의 사진 보관함에 저장하는 기능을 제공합니다. 이 이미지는 기기 안에서만 생성·저장되며 외부로 전송되지 않습니다. 당사 앱은 사진 보관함에 이미 저장된 다른 사진을 읽거나 수집하지 않으며, 저장 권한은 오직 앱이 만든 이미지를 추가하기 위해서만 사용됩니다.

iOS에서는 앱이 배경화면을 직접 변경할 수 없으므로, 저장된 이미지를 사용자가 설정 → 배경화면에서 직접 선택해 적용하게 됩니다. 이 과정에서 당사 앱이 수집하는 정보는 없습니다.

3. 광고를 통한 자동 수집 정보

당사 앱은 무료로 제공되며, 서비스 운영을 위해 Google AdMob 광고를 표시합니다. AdMob은 광고 제공 및 측정을 위해 다음과 같은 정보를 자동으로 수집·처리할 수 있습니다.

  • 광고 식별자 (iOS: IDFA / Android: 광고 ID)
  • IP 주소
  • 기기 정보 (기기 모델, 운영체제 버전, 언어 설정 등)
  • 광고 상호작용 데이터 (광고 노출, 클릭 등)
  • IP 기반의 대략적인 위치 정보

이 정보는 Google로 전송되며, 처리 방식에 관한 자세한 내용은 Google 개인정보처리방침을 참조해 주세요: https://policies.google.com/privacy

iOS 기기에서는 설정 → 개인정보 보호 및 보안 → 추적에서 앱의 추적 요청을 거부할 수 있으며, Android 기기에서는 설정 → Google → 광고에서 광고 식별자를 재설정하거나 맞춤 광고를 끌 수 있습니다.

4. 앱이 요청하는 권한

권한사용 목적
사진 추가 (iOS: NSPhotoLibraryAddUsageDescription / Android: 저장소) 사용자가 만든 할 일 이미지를 사진 보관함에 저장하기 위해 사용합니다. 기존 사진을 읽거나 수집하지 않습니다.
인터넷 (INTERNET) 광고 로딩을 위해 사용합니다.
광고 ID (iOS: App Tracking Transparency / Android: AD_ID) Google AdMob 광고 제공 및 측정을 위해 광고 식별자에 접근합니다. 사용자가 거부할 수 있으며, 거부해도 앱의 모든 기능을 이용할 수 있습니다.

5. 데이터 보관 및 삭제

당사 앱이 저장하는 모든 사용자 데이터(할 일, 메모, 설정 등)는 사용자 기기 내부에만 저장됩니다. 사용자가 앱을 삭제하면 모든 로컬 데이터도 함께 삭제됩니다. 단, 사진 보관함에 이미 저장한 이미지는 사용자의 사진으로 남으며 직접 삭제할 수 있습니다. 당사 앱은 자체 서버에 사용자 데이터를 보관하지 않으므로 별도의 서버 측 삭제 절차가 없습니다. Google AdMob이 수집한 데이터의 보관 기간 및 삭제 정책은 Google 개인정보처리방침을 따릅니다.

6. 데이터 보안

사용자 콘텐츠는 사용자 기기 내 로컬 저장소에만 저장됩니다. 광고 로딩 등 외부 통신은 표준 보안 프로토콜(HTTPS)을 통해 이루어집니다.

7. 아동 개인정보 보호

당사 앱은 만 13세 미만 아동을 주요 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

8. 개인정보처리방침의 변경

본 개인정보처리방침은 법령 변경 또는 서비스 변경에 따라 수정될 수 있으며, 변경 시 본 페이지를 통해 공지합니다.
