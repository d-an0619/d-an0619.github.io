데이코스(DayCourse) 개인정보처리방침

최종 업데이트: 2026년 7월 4일

데이코스(DayCourse, 이하 "당사 앱")는 사용자의 개인정보를 소중하게 다루며, 관련 법령에 따라 다음과 같이 개인정보를 처리합니다. 당사 앱은 하루 동선·방문 순서를 정리하는 코스 플래너로, 별도의 회원가입·로그인 없이 사용됩니다.

1. 당사 앱이 직접 수집·저장하는 정보

당사 앱은 사용자의 개인정보를 자체 서버로 수집하거나 전송하지 않습니다. 당사 앱은 자체 서버를 운영하지 않으며, 회원가입·로그인·클라우드 동기화 기능을 제공하지 않습니다. 사용자의 이름, 이메일, 전화번호, 계정 정보를 수집하지 않습니다.

사용자가 앱 내에서 만든 코스, 장소, 방문 시간, 메모 등 모든 사용자 콘텐츠는 사용자 기기 내부의 로컬 데이터베이스(SQLite)에만 저장됩니다.

2. 제3자 서비스로 전송되는 정보

당사 앱은 지도 표시와 장소 검색 기능을 제공하기 위해 다음 외부 서비스를 이용하며, 기능 수행에 필요한 최소한의 정보가 해당 서비스로 전송됩니다. 각 정보는 요청 처리에만 사용되며, 당사 앱이 별도로 저장하지 않습니다.

서비스목적전송되는 정보
네이버 지도(NAVER Maps SDK) 지도 표시 및 현재 위치 표시 지도 화면 좌표, 사용자 위치(사용 시)
카카오 로컬(Kakao Local REST API) 장소 검색 및 주소 조회 사용자가 입력한 검색어, 검색 기준 좌표

각 서비스의 개인정보 처리에 관한 자세한 내용은 아래 정책을 참조해 주세요.
네이버: https://www.navercorp.com/utility/privacy
카카오: https://www.kakao.com/policy/privacy

3. 광고를 통한 자동 수집 정보

당사 앱은 무료로 제공되며, 서비스 운영을 위해 Google AdMob 광고(네이티브·전면 광고)를 표시합니다. AdMob은 광고 제공 및 측정을 위해 다음과 같은 정보를 자동으로 수집·처리할 수 있습니다.

  • 광고 식별자 (Advertising ID, AAID)
  • IP 주소
  • 기기 정보 (기기 모델, 운영체제 버전, 언어 설정 등)
  • 광고 상호작용 데이터 (광고 노출, 클릭 등)
  • IP 기반의 대략적인 위치 정보

이 정보는 Google로 전송되며, 처리 방식에 관한 자세한 내용은 Google 개인정보처리방침을 참조해 주세요: https://policies.google.com/privacy

Android 기기에서 광고 식별자는 설정 → Google → 광고에서 재설정하거나 맞춤 광고를 끌 수 있습니다.

4. 앱이 요청하는 권한

권한사용 목적
위치 (ACCESS_FINE_LOCATION / ACCESS_COARSE_LOCATION) 지도에 사용자의 현재 위치를 표시하고, 장소 검색의 기준 위치로 사용합니다. 위치 정보는 기기와 지도 서비스에서만 사용되며 당사 서버에 저장되지 않습니다. 권한 허용은 선택 사항이며, 거부해도 코스 작성 기능은 이용할 수 있습니다.
사진·미디어 저장 (WRITE_EXTERNAL_STORAGE) 사용자가 완성한 코스를 이미지로 저장하기를 선택할 때, 해당 이미지를 기기 갤러리에 저장하기 위해 사용합니다. 기존 사진·미디어를 읽거나 수집하지 않습니다.
인터넷 (INTERNET) 지도 타일 로딩 및 장소 검색 요청을 위해 사용합니다.
광고 ID (com.google.android.gms.permission.AD_ID) Google AdMob 광고 제공 및 측정을 위해 광고 식별자에 접근합니다.

5. 데이터 보관 및 삭제

당사 앱이 저장하는 모든 사용자 데이터(코스, 장소, 메모 등)는 사용자 기기 내부에만 저장됩니다. 사용자가 앱을 삭제하면 모든 로컬 데이터도 함께 삭제됩니다. 당사 앱은 자체 서버에 사용자 데이터를 보관하지 않으므로 별도의 서버 측 삭제 절차가 없습니다. Google AdMob이 수집한 데이터의 보관 기간 및 삭제 정책은 Google 개인정보처리방침을 따릅니다.

6. 데이터 보안

사용자 콘텐츠는 사용자 기기 내 로컬 저장소에만 저장됩니다. 지도·검색 기능 이용 시 전송되는 정보는 각 서비스와의 통신에만 사용되며, 통신은 표준 보안 프로토콜(HTTPS)을 통해 이루어집니다.

7. 아동 개인정보 보호

당사 앱은 만 13세 미만 아동을 주요 대상으로 하지 않으며, 의도적으로 아동의 개인정보를 수집하지 않습니다.

8. 개인정보처리방침의 변경

본 개인정보처리방침은 법령 변경 또는 서비스 변경에 따라 수정될 수 있으며, 변경 시 본 페이지를 통해 공지합니다.
